MATTIS DA COSTA FARO — PORTFOLIO
=================================

Structure du dossier :
- index.html      → Site principal one-page
- 404.html        → Page d'erreur 404
- Assets/
  └── Images/
      └── Mattis.jpg   ← Placer votre photo ici

Installation :
1. Dézipper le dossier
2. Placer votre photo dans Assets/Images/Mattis.jpg
3. Ouvrir index.html dans votre navigateur
4. Remplacer les liens GitHub, LinkedIn, Twitter, Email par les vôtres

Contact form :
- Utilise Formspree (action="https://formspree.io/f/xaqdkzlw")
- Créer un compte sur formspree.io et remplacer l'ID si besoin
